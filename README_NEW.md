# 赛博疗愈师 (Cyber Healer)

**版本：** V1.0 MVP
**日期：** 2025年12月29日

---

## 产品综述

### 核心价值主张
利用 Gemini 3 Pro 的超高情商推理与原生多模态能力，将用户的"八字数据"翻译为"心理学图谱"。

- **去迷信化：** 不预测吉凶，只分析能量结构与性格底色
- **情绪变现：** 提供极具共情力的文字抚慰和视觉化的能量图腾
- **低门槛介入：** 用"玄学"作为趣味入口，交付"心理疗愈"的内核

---

## 核心功能

### 1. 授权码验证系统
- 用户首次访问需输入授权码
- 验证通过后永久生效（本地存储）
- 每天免费 5 次预测

### 2. 用户信息输入
- 出生年月日时（支持时间不详选项）
- 性别选择
- 自动进行八字排盘计算

### 3. AI 报告生成
基于 Gemini 3 Pro 生成个性化疗愈报告，包含：
- **能量原型：** 用具体意象定义用户
- **光与影：** 天赋与成长课题
- **当下回应：** 心理疏导
- **行动处方：** 3个具体可执行的建议

---

## 技术栈

- **前端：** React + Vite + TypeScript + Tailwind CSS
- **后端：** Supabase (数据库 + Edge Functions)
- **AI引擎：** Gemini 3 Pro API
- **命理计算：** lunar-typescript (八字排盘库)

## 核心特性

### 授权码验证流程
1. 用户首次访问进入授权页面
2. 输入授权码进行验证
3. 验证成功后保存到 localStorage
4. 后续访问自动跳过验证，直接进入主功能

### 简化的用户体验
- 去除了复杂的情绪文本输入
- 用户只需填写基本信息即可
- 系统自动生成通用的分析请求
