# 🚀 快速开始指南

## 📋 功能概览

赛博疗愈师现在包含以下核心功能：

1. **授权码验证** - 首次访问需要验证，之后永久生效
2. **简化输入** - 只需填写生日、性别等基本信息
3. **AI 分析** - 基于 Gemini 3 Pro 生成个性化疗愈报告

---

## 🎯 使用流程

### 第一步：启动项目

```bash
# 进入项目目录
cd app-8fg4m9n7c0e9

# 安装依赖（首次运行）
npm i

# 启动开发服务器
npm run dev -- --host 127.0.0.1
```

访问：`http://127.0.0.1:5173`

---

### 第二步：输入授权码

首次访问会看到授权码验证页面：

![授权页面示意]

**测试授权码（任选其一）：**
- `CYBER2025`
- `HEALER888`
- `TEST123`

输入后点击"确认授权"按钮。

✅ **验证成功后，授权码会自动保存，下次访问无需再次输入！**

---

### 第三步：填写基本信息

进入主页面后，填写以下信息：

1. **称呼**（可选）- 如何称呼你
2. **降落日期** - 出生年月日
3. **时刻坐标** - 出生时辰（可勾选"时间不详"）
4. **能量流向** - 性别选择

**注意：** 
- 不再需要填写"发射信号"（情绪文本）
- 系统会自动进行通用分析

---

### 第四步：生成报告

点击"开始链接"按钮，等待 10-20 秒。

系统会生成包含以下内容的疗愈报告：

- 🌟 **能量原型** - 用意象定义你的特质
- ☀️ **光与影** - 你的天赋与成长课题
- 💬 **当下回应** - 针对性的心理疏导
- 📝 **行动处方** - 3个具体可执行的建议

## 💡 常见问题

### Q1: 授权码在哪里获取？
A: 目前使用测试授权码：`CYBER2025`、`HEALER888`、`TEST123`

### Q2: 授权码会过期吗？
A: 授权码本身有有效期，但验证通过后会保存到本地，无需重复验证。

### Q3: 如何清除授权码？
A: 打开浏览器开发者工具（F12），在 Console 中输入：
```javascript
localStorage.removeItem('cyber_healer_auth_code')
```
然后刷新页面。

### Q4: 为什么去掉了情绪文本输入？
A: 为了降低使用门槛，让用户更轻松地使用系统。系统会自动使用默认的分析请求。

### Q5: 生成报告需要多久？
A: 通常需要 10-20 秒，取决于 Gemini API 的响应速度。

## 🔧 技术说明

### 授权码存储
- 使用 `localStorage` 存储
- Key: `cyber_healer_auth_code`
- 验证通过后自动保存

### 路由结构
- `/` - 授权码验证页面
- `/home` - 主功能页面

### API 调用
- 验证授权码：Supabase RPC `validate_redeem_code`
- 生成报告：Supabase Edge Function `generate-report`
